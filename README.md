## Description
A digital sign of the current fire danger rating in the community of Walnut Mountain, Elijay, GA.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# end to end tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
